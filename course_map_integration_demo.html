Funkcje Mapy Kursu

🗺️ Mapa Kursu

Wizualna reprezentacja hierarchii: Kurs → Moduły → Kategorie → Lekcje. Interaktywne węzły z kolorami charakterystycznymi dla każdego modułu.

📊 Statystyki

Przegląd liczby modułów, kategorii i lekcji. Informacje o postępie użytkownika i estymowanym czasie ukończenia.

🎯 Umiejętności

Istniejący system umiejętności z opcjami filtrowania i sortowania według różnych kryteriów.

🎮 Interaktywność

Możliwość poruszania węzłów, zbliżania/oddalania, animacje fizyczne i responsywny design.

🎨 Personalizacja

Kolorowanie węzłów według postępu, różne rozmiary węzłów według ważności, opcje uproszczonego widoku.

📱 Responsywność

Dostosowanie do różnych rozmiarów ekranów, optymalizacja dla urządzeń mobilnych i tabletów.

📁 Zmodyfikowane Pliki

Jak Używać

1. Uruchomienie Aplikacji

streamlit run main.py

2. Nawigacja

  1. Zaloguj się do aplikacji BrainVenture Academy
  2. Przejdź do sekcji "Umiejętności" w menu głównym
  3. Wybierz zakładkę "🗺️ Mapa Kursu"
  4. Eksploruj interaktywną mapę struktury kursu

3. Opcje Wizualizacji

Pełna Struktura

Pokazuje wszystkie poziomy: Kurs → Moduły → Kategorie → Lekcje (150+ węzłów)

Uproszczona Mapa

Fokus na główną strukturę: Kurs → Moduły → Kategorie (20 węzłów)

4. Interakcje

  • Przeciąganie - Przesuń węzły aby zmienić układ
  • Zoom - Scrolluj aby przybliżyć/oddalić mapę
  • Kliknięcie - Wybierz węzeł aby go wyróżnić
  • Animacje - Obserwuj dynamiczne układy fizyczne

🎯 Następne Kroki

Po uruchomieniu aplikacji, mapa kursu będzie dostępna w zakładce Umiejętności. Użytkownicy mogą eksplorować pełną strukturę kursu w sposób wizualny i interaktywny.
